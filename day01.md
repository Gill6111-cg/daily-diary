# 🚀 MERN Stack Training Diary – Day 01
**Date:** 22 June 2026

## 📚 Topic: Introduction to HTML and Basic Webpage Creation

Today was the first day of my MERN Stack training. I was introduced to HTML (HyperText Markup Language), which is the foundation of web development and is used to create the structure of web pages. I learned that every website we visit is built using HTML along with CSS and JavaScript. During the session, I created my first webpage and explored different HTML tags and their uses.



## 💻 Topics Covered

### 1. HTML Document Structure
I learned the basic structure of an HTML document. Every HTML page starts with `<!DOCTYPE html>`, which tells the browser that the document is an HTML5 file. The `<html>` tag contains the entire webpage. Inside it, the `<head>` section stores information about the webpage, such as the title and character encoding, while the `<body>` section contains all the content that is visible to users.

### 2. Headings and Text Formatting
I used the `<h1>` tag to create a heading on my webpage. I also learned text formatting tags such as:
- `<b>` for making text bold
- `<i>` for displaying text in italic style
- `<u>` for underlining text
- `<br>` for moving content to the next line

These tags help in presenting content in a more readable and attractive way.

### 3. Images and Alignment
I learned how to display an image on a webpage using the `<img>` tag by specifying the image source, height, and width. I also centered the image on the page to understand how elements can be aligned and positioned.

### 4. Buttons and Hyperlinks
I created a button using the `<button>` tag and learned that buttons are used to perform actions when clicked. I also added a hyperlink using the `<a>` tag, which allows users to navigate from one webpage to another.

### 5. Lists
I learned how to organize data using lists:
- **Unordered List (`<ul>`):** Displays items with bullet points. I used square bullets for my list.
- **Ordered List (`<ol>`):** Displays items in a sequence. I practiced lists with alphabetical numbering and Roman numerals.

Lists make information more structured and easier to understand.

### 6. Tables
I created a simple table using:
- `<table>` to create the table
- `<tr>` to create rows
- `<th>` to create table headings
- `<td>` to insert data into table cells

Tables are useful for presenting information in rows and columns.

### 7. Forms
I created a basic form using:
- `<form>` to define the form section
- `<label>` to provide names for input fields
- `<input type="text">` to allow users to enter data

I learned that forms are important because they allow users to provide information such as names, emails, passwords, and other details on websites.

## 🎯 Key Learning Outcomes
- Understood the basic structure of an HTML document.
- Learned how to use different HTML tags to create and organize webpage content.
- Gained hands-on experience by building my first webpage from scratch.
- Developed a basic understanding of how websites are structured and displayed in a browser.
